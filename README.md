# Hotel-Bookings-Dashboard-with-Power-BI
## Project Overview
This project presents an interactive Hotel Bookings Dashboard developed using Power BI to analyze and visualize hotel reservation data. The dashboard provides actionable insights into booking trends, customer behavior, and revenue performance to support data-driven decision-making.
